# Data Sources

This project utilizes publicly available transportation datasets from the Florida Department of Transportation (FDOT).

Datasets currently collected include:

- Linear Referencing System (LRS) routes
- Annual Average Daily Traffic (AADT)
- Route centerlines
- Road arcs
- Mile markers
- Speed limits
- Interchanges
- Ramps
- Traffic signal locations
- Railroad crossings
- Bike lanes
- Sidewalk information

Due to dataset size and update frequency, source datasets may not be stored directly in the repository.
